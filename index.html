wine-deep
One-command Wine dependency installer for Linux & macOS

⚡ Install

curl -fsSL https://ardakircar.github.io/Wine-Deep/install.sh | bash

What it installs

wine-deep silently installs the following winetricks libraries into your existing Wine prefix. No Microsoft popups. Wine Mono is preserved.

d3dcompiler_42
d3dcompiler_43
d3dcompiler_47
d3dx11_43
d3dx9
dxvk
quartz
vkd3d
vcrun2019 (silent)
⚠️ vcrun2019 is installed silently using WINEDLLOVERRIDES to prevent the Microsoft .NET installer from replacing Wine Mono. This keeps your Wine prefix stable.

Requirements

  • Wine must already be installed
  • winetricks (auto-installed if missing)
  • Linux (apt / pacman / dnf) or macOS (Homebrew)

How it works

The script detects your OS, checks for Wine and winetricks, then installs each package using winetricks -q (quiet/unattended mode). At the end it prints a summary of what succeeded and what (if anything) needs manual attention.
